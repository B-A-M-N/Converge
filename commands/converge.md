---
description: Manage Converge recurring tasks
---

Converge provides commands to create, manage, and monitor recurring jobs.

Available commands:
- `/converge-add` - Create a new recurring task
- `/converge-ls` - List all jobs
- `/converge-status <job-id>` - Get job details
- `/converge-pause <job-id>` - Pause a job
- `/converge-resume <job-id>` - Resume a paused job
- `/converge-cancel <job-id>` - Delete a job
- `/converge-run-now <job-id>` - Trigger immediate execution
- `/converge-logs <job-id>` - View run history
- `/converge-doctor` - Check system health

Run any command with `--help` (e.g., `/converge-add --help`) to see its arguments.
